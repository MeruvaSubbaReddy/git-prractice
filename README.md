git --version  for version of git
git init    to intialize git in m project folder
touch filename  to create any type of file
mkdir foldername to create folder
git add   this is for   staging the data
git commit   this is for commit our data
git rm --cached file name   to remove staging data
git branch branchName  to create new branch
git checkout branchName  to navigate one branch to another branch
git merge branch   this is for merging our child branch to master branch
git remote add origin "url" to create origin
git remote  this is for checking origin is created or not
git push -u origin master   this is for transfer our local repo to global repo
touch .gitignore   this file is to untrack unwanted files in git
